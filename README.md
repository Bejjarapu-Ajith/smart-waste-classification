# Smart Waste Classification System

## Overview

Smart Waste Classification System is an AI-based computer vision project developed to classify waste into Dry Waste and Wet Waste categories.

The goal of the project is to support smarter waste management and encourage proper waste segregation using deep learning and real-time prediction techniques.

The system combines model training, prediction, and performance analysis into a complete workflow built using Python and modern AI tools.

---

## Features

* AI-powered dry and wet waste classification
* Deep learning-based model training
* Real-time prediction capability
* Streamlit-based interactive interface
* Performance evaluation using training metrics and graphs
* Organized storage for trained models and results

---

## Technologies Used

* Python
* PyTorch / YOLO
* OpenCV
* Streamlit
* NumPy
* Matplotlib

---

## Project Structure

* `app.py` → Streamlit application for waste prediction
* `data.yaml` → Dataset configuration file
* `Trained models/` → Saved model weights
* `metrics/` → Training graphs and evaluation metrics
* `Screenshots/` → Project output screenshots

---

## Performance Metrics

The project includes several evaluation metrics to measure model performance:

* Confusion Matrix
* Precision-Recall Curves
* F1 Score Curves
* Training Result Graphs
* Validation Metrics

These metrics help analyze the accuracy and effectiveness of the waste classification model.

---

## How to Run the Project

### 1. Install Required Libraries

```
pip install -r requirements.txt
```

### 2. Launch the Streamlit Application

```
streamlit run app.py
```
